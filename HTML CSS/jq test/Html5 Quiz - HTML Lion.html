HTML5 Quiz

  • which of the following is not true about aside element?

    Your Answer: Any element that may contain text or embeded content may contain this element
    Correct Answer: tabindex attribute can not be used with this element

  • Which of the following browser may not support the following code to create a rectangle?
    <!doctype html><html><body><canvas id="myCanvas" width="350" height="200" style="border:1px solid red;">Your browser does not support the HTML5 canvas tag.</canvas><body><html>

    Correct Answer: Internet Explorer 7.0

  • Which tag is used to contain all the information at the top of your page? (logo, login, etc.)

    Correct Answer: <header></header>

  • Which tag is used to contain all the information at the bottom of your page? (copyright, quick links, etc.)

    Your Answer: <header></header>
    Correct Answer: <footer></footer>

  • Which tag is used to contain your navigation?

    Correct Answer: <nav></nav>

  • The <article> is used to contain a main article. What is the tag used to break it into sections?

    Correct Answer: <section></section>

  • What tag do you use to mark up any dates or time?

    Correct Answer: <time></time>

  • Which tag is used to mark up data that is related to the story but not part of the flow? (ex. side notes)

    Correct Answer: <aside></aside>

  • Which tag do you use to keep heading tags together if needed?

    Correct Answer: <hgroup></hgroup>

  • The <figure> tag allows you to put an image for a diagram or example for an article. Which tag allows you to put a caption for that image?

    Correct Answer: <figcaption></figcaption>

  • Which method is used to stop web worker?

    Your Answer: worker.stop()
    Correct Answer: w.terminate()

  • Which of these is not valid HTML5 section element?

    Your Answer: <header></header>
    Correct Answer: <body></body>

  • Which of the following is not media element?

    Your Answer: <embed></embed>
    Correct Answer: <readyToPlay></readyToPlay>

  • Which of the elements are supported in mobile platform in HTML5?

    Your Answer: 3D canvas
    Correct Answer: None

  • Which tag of HTML5 is not supported by any browser?

    Correct Answer: track

  • HTML5 provides a standard for showing video on a web page. Select the correct element to achieve this.

    Correct Answer: <video src="filename.ext" controls="controls"> </video>

  • Select the correct element for defining a navigation menu for your site.

    Correct Answer: <nav></nav>

  • Which of the following elements are no longer supported in HTML5

    Correct Answer: <acronym></acronym>

  • In html5 SSE stands for...

    Your Answer: Server side event
    Correct Answer: Server sent event

  • How can you specify default text in an input field?

    Correct Answer: Using the 'placeholder' attribute
